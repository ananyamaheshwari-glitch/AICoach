FREQUENTLY ASKED QUESTIONS
================================================================================

Q: Do I need to install Git?
A: Yes, follow steps in GITHUB_DEPLOYMENT.md to install for Windows.

Q: What's the GitHub URL for my project?
A: https://github.com/YOUR_USERNAME/ai-quiz-app (replace YOUR_USERNAME)

Q: Can I deploy this to production immediately?
A: Not recommended. Complete Phase 1 security fixes first (see SECURITY_REVIEW.md).

Q: How do I add more quiz questions?
A: Edit backend/db/database.js seedQuestions array and restart.

Q: The Hugging Face API is slow. What should I do?
A: Normal first time. LLM responses take 5-15 seconds. See SECURITY_REVIEW.md
   for caching recommendations.

Q: Where do I find my API errors?
A: Check terminal where backend is running, or browser console (F12).

Q: Can I change the quiz topic?
A: Yes, edit database.js and add new topics to questions table.
